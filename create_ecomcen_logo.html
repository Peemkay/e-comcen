  4. Move the downloaded files to these exact locations:

    • C:\Users\chaki\Documents\Projects Augment\NASDS\assets\images\nas_icon.png

    cd "C:\Users\chaki\Documents\Projects Augment\NASDS"
flutter pub run flutter_launcher_icons
  6. If you encounter the "NoDecoderForImageFormatException" error, try:
    • Opening the downloaded images in Paint or another image editor

    • Then placing them in the assets/images directory
